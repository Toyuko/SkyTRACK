# Tauri Icons

Place your app icons here. Tauri requires the following icon files:

- `32x32.png` - 32x32 pixel PNG
- `128x128.png` - 128x128 pixel PNG
- `128x128@2x.png` - 256x256 pixel PNG (for Retina displays)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

For now, Tauri will use default icons if these files are missing. You can generate icons later using tools like:
- [Tauri Icon Generator](https://github.com/tauri-apps/tauri-icon)
- Online icon generators
- Design tools like Figma, Sketch, etc.
